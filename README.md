collection of platformio scripts and projects

`brew install pio`

# adding a a project
- `mkdir myProject`
- `pio boards 'my board'`
- `pio init --board 'my board'`

# other commands

```
pio run  # compile
pio target --upload  # flash
pio lib search 'some lib' # search libs
pio lib install 'some lib' # install libs
```
